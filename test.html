3 시나리오 테스트
QR 자산인식
회의실 프로젝터 고장 신고
QR 스캔 → 프로젝터 자동 인식 → "전원 안 됨" 선택 → 설명 작성 → 접수
 카테고리 선택
냉방 온도 불만 접수
카테고리 "환경" 선택 → "냉방" → 위치 수동 입력 → 설명 작성 → 접수
 익명 신고
화장실 청소 상태 익명 신고
익명 모드 ON → 시설 → "화장실 청소" → 접수 (신고자 비공개)
 관리자
접수 건 상태 변경 + 자산 점검일정 확인
대시보드에서 상태 드롭다운 변경 → 자산관리 탭 → 점검일정 탭 → 교체이력
 사용자
내 신고 현황 확인 + 상세 이력
필터 전환 (전체/접수/처리중/완료) → 카드 클릭 → 타임라인 확인
